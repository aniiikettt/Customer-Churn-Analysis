# 🚀 Customer Churn Analysis Project

## 📌 Problem Statement

Customer churn is one of the biggest challenges faced by subscription-based businesses. Losing customers not only affects revenue but also increases the cost of acquiring new users. However, many organizations struggle to understand *why* customers leave and *which factors* contribute the most to churn.

Without clear insights, decision-making becomes reactive instead of proactive.

---

## 🎯 What I Aim to Solve

This project focuses on analyzing customer data to uncover patterns behind churn behavior. The goal is to:

- Identify key factors influencing churn  
- Understand customer segments at higher risk  
- Analyze how pricing, tenure, services, and contracts impact retention  
- Transform raw data into actionable business insights  

---

## 🛠️ Approach & Methodology

To solve this problem, I used a structured data analysis pipeline:

### 1. Data Processing (Python)
- Cleaned and preprocessed the dataset using Pandas  
- Handled missing values and converted data types  
- Created meaningful features like customer segments and charge groups  

### 2. Data Analysis (SQL)
- Performed queries to analyze churn distribution  
- Identified trends across gender, contract type, and services  
- Calculated key metrics like churn rate and averages  

### 3. Data Visualization (Power BI)
- Built an interactive dashboard to visualize insights  
- Highlighted patterns in churn behavior  
- Enabled easy interpretation for decision-making  

---

## 📊 Key Insights

- Customers with **month-to-month contracts** show higher churn  
- **Higher monthly charges** are linked with increased churn risk  
- **New customers (low tenure)** are more likely to leave  
- Specific service issues and competitor offerings drive churn  

---

## ✅ Solution Outcome

By combining Python, SQL, and Power BI, this project transforms raw customer data into meaningful insights. It helps businesses:

- Identify high-risk customers early  
- Improve retention strategies  
- Make data-driven decisions  
- Reduce churn and increase customer lifetime value
